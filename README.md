## 👋 Welcome

Welcome to my dedicated repository for the **#DrGViswanathan Challenge**.

Over the next 50 days, I am committing to a rigorous daily routine focused on strengthening my programming fundamentals. This initiative, driven by the VIT Bhopal Placement Office, is strategically designed to cultivate robust problem-solving skills, ensure consistent coding habits, and build a powerhouse GitHub profile to unlock top-tier tech opportunities.

---

## 🎯 Objectives

- **Master Fundamentals:** Deep dive into basic coding, foundational mathematics, and Data Structures & Algorithms (DSA).
- **Maintain Consistency:** Develop a disciplined, daily approach to technical problem-solving.
- **Track Growth:** Document progress transparently to showcase readiness for technical interviews and industry challenges.

---

## 📌 Challenge Guidelines & Methodology

- ⏱️ **Consistency is Key:** Dedicating focused time every single day to tackle new logic and math problems.
- 💻 **Language Choice:** All solutions are strictly implemented in **C++**. This ensures the focus remains entirely on core logic, memory management, and algorithmic efficiency.
- 📢 **Visibility & Accountability:** Every completed challenge is pushed to this repository and shared on **LinkedIn** using the `#DrGViswanathan` hashtag to demonstrate ongoing progress and accountability to recruiters.
